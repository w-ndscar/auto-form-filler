# auto-form-filler
An auto form filler for an internal web application named ESMT

*Pre-run check*: Select the correct project and ensure all elements are present. If any are missing, create them before running the automation — otherwise, the 'Elements' section will appear empty.

1. Click 'Open Chrome' and log in to the website
2. Select the project and go to the Drawings tab
3. Use 'Browse' to select your Excel file
4. Enter the sheet name and ending row number
5. Click 'Start Automation' to begin

**Note**: After entering the values, it'll wait.
Press 'Esc' key after each entry to proceed to the next. DO NOT click Submit/New Buttons. It will be done automatically.
